Chatting - 💬 Real-Time Chat Application (MERN Stack)
A full-stack real-time chat application built using the MERN stack with secure authentication and instant messaging capabilities.

Features
🔐 User Authentication (JWT & bcrypt)
💬 Real-time messaging using Socket.io
👥 Multi-user chat support
📱 Fully responsive UI
⚡ Fast and optimized state management
🔒 Secure password hashing
🟢 Online/Offline user status (if implemented)

Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Token), bcrypt.js
Real-time Communication: Socket.io


How it works
Users register/login securely using JWT authentication.
Passwords are hashed using bcrypt before storing in database.
Socket.io establishes a real-time connection between clients.
Messages are sent instantly without page reload.
Backend handles API requests and database operations.
